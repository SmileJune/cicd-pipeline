## CI/CD 프로세스 설계 과제

간단한 C 프로그램을 이용하여 전체 프로세스와 내부 파이프 라인을 구축하라.

**************************구현 예정 리스트**************************

1. 브랜치 관리
    1. main - 운영 서버 코드
    2. develop - 개발 서버 코드
    3. feature - 개발 중인 코드
2. PR시 Lint 체크
3. PR시 Unit Test 및 Test Report
4. MERGE시 브랜치에 맞는 서버에 배포
5. 이슈 발생 시 이전 버전으로 다시 배포하는 페일오버 전략 구축
